OPERATIONS, MAINTENANCE and CALIBRATION for COBAS 6000

  1. PURPOSE
    1. Cobas 6000 is a fully automated, computerized chemistry analyzer. It has 2 modules : e601 and c501. Cobas 6000 is intended for in vitro quantitative or qualitative determination of a wide range of analytes in various body fluids such as classical chemistry, electrolytes, specific proteins, therapeutic drug monitoring, drugs of abuse, and hormone testing which are consolidated into one system with two reagent cassette design.
  2. PRINCIPLE
    1. The c 501 module comprises a photometric unit and an ISE unit (for ion-selective electrode [ISE] determinations).
    2. The e 601 module is a multi-test immunoassay analyzer with random access with a capacity of up to 170 tests per hour. The cobas® 6000 analyzer series can be configured with up to two e 601 modules.
    3. 2.3. The graphical user interface provides quick and easy access to sample, control, and calibration data, while continuously monitoring all system functions. Color-coded icons alert the user to changes in the system status. Connection to the host system allows for automatic transfer of results to and from the analyzer.

  10. SAFETY PRECAUTIONS
    1. Do not remove any cover of the instrument, other than specified in the Operator’s Manual.
    2. Do not open the top cover and touch the ultrasonic mixing unit during operation or when the analyzer performs maintenance.
    3. Be sure to wear protective equipment.
    4. If any biohazardous material is spilled, wipe it up immediately and apply disinfectant.
    5. If waste solution comes into contact with your skin, wash it off immediately with water and apply a disinfectant. Consult a physician.
    6. Do not touch the cooling fan during operation
    7. Before cleaning the fan, be sure to shutdown the instrument.
    8. Before starting operation or maintenance, be sure to close and lock the top cover.
    9. Observe the cautions given on the bottles and cassettes and the instructions for use
    10. Refer to the safety precautions given in the PHDB1 Safety Manual.
  11. PROCEDURES
    1. The analyzer can be started manually or automatically. The analyzer (c 501, e 601, and control unit) performs initialization and then enters standby mode ( time approximately 12 min).
      1. To automatically start up the analyzer:
        1. The analyzer can be programmed to start up automatically. If the analyzer was put in sleep mode at the end of the previous working session and a wake-up time was set.
        2. The analyzer starts up automatically at the set time. While the analyzer is performing initialization, the log-on screen is displayed.
        3. Enter your Operator ID and password to log on.
        4. Choose OK to gain access to the software and begin operation. When initialization is completed, the analyzer goes into standby.
      2. To manually start up the analyzer:
        1. on the operation power switch, located on the left side of the rack sampler unit.
        2. Switch on each power switch of the computer, printer, and monitor.
        3. Enter your Operator ID logon and password to log on.
        4. Choose OK to gain access to the software and begin operation. When initialization is completed, the analyzer goes into standby.
    2. Check Systems Alarm:
      1. If an alarm was issued, the Alarm global button flashes. In this case, it is necessary to open the Alarm screen to view the alarm. The Alarm screen identifies any system alarm conditions.
      2. Choose Alarm (global button) to display the Alarm screen.
      3. Select each alarm to view the description and remedies (displayed in the lower half of the screen).
      4. Correct any alarm conditions by following the remedies.
      5. Choose Close to close the Alarm screen.
    3. Print reagent load list:
      1. Choose Reagent Preparing on the System Overview screen.
      2. Choose Reagent Load List on the Reagent Preparing window. A confirmation window appears.
      3. Choose Yes to print the Reagent Load List/Unload List.
    4. To replace ISE IS or ISE Dil:
      1. Remove the empty bottle from the position where the green indicator lamp lights up.
      2. Place a new bottle into the reagent position. Reagent registration is performed automatically when a new bottle is placed.
    5. To load a cobas c pack:
      1. Fold the cassette table at the front of the c 501 module downward to get access to the cassette loading port.
      2. Place the cobas c pack right in front of the loading port with its barcode label facing to the right.
      3. Slide the cobas c pack all the way into the loading port until you sense a resistance.
    6. To load cobas e packs:
      1. Make sure, that the system is in standby.
      2. Lift off the lid of the reagent disk.
      3. Place the required reagents in the reagent disk. Ensure that the cobas e packs are placed in the reagent disk in the correct orientation.
      4. Close the lid of the reagent disk. A reagent scan is activated and the Reagent Overview window (System Overview > Reagent Overview) is updated.
    7. To load patient samples:
      1. Prepare sample racks with the correct color and rack numbers according to sample types (erum, Urine, Plasma, Supernatant).
      2. For all routine samples use gray racks.
      3. To verify the correct rack numbers for each sample type refer to the Rack Assignment area on the Utility > System screen.
      4. Place the samples in the prepared sample racks. Ensure the sample barcodes are facing the open slot in the rack so the barcode reader can scan them.
      5. Place the routine sample racks onto a rack tray in the correct orientation: The barcode labels of the sample racks have to be on the right side when placing the tray into the rack loader. The rack loader is on the left side.
      6. Before starting a run, ensure that all test selections have been made and all necessary samples, calibrators, and controls have been loaded.
      7. Choose Start (global button).
      8. Verify the settings on the Start Conditions screen.
      9. Choose Start (on the Start Conditions screen). The analyzer performs preparation routine and then begins to process the samples.
      10. Test selection for STAT samples is the same as for routine samples. That is, test selections for samples are downloaded from the Host in realtime but can still be changed from the analyzer’s control unit.
      11. To load STAT samples:
        1. Prepare sample racks with the correct color and rack numbers according to sample types.
        2. Set the samples in the prepared sample racks. Ensure the sample barcodes are facing the open slot in the rack so the barcode reader can scan them.
        3. Place the STAT rack in the STAT port carefully observing its correct orientation.
    8. Checking results:
      1. As results are generated on the instrument, they are saved in the database located on the internal hard disk of the control unit. When all results for all the tests requested for a sample are available, the analyzer sends them to the Host where they can be validated.
      2. To view or edit results on the analyzer, use the Workplace > Data Review screen; all test results in the database are displayed here.
    9. To activate the automatic rerun function:
      1. Verify the settings of the Automatic Rerun check box on Utility > Application > Range.
      2. Choose Start (global button).
      3. Choose Change in the Automatic Rerun area.
      4. Choose the Routine check box, Stat check box, or both to process reruns without operator intervention. Routine and STAT reruns are selected separately.
    10. Shutting down the analyzer:
      1. Choose Shut Down (global button) to display the Shut Down window.
      2. Select the Shut Down option, choose Execute and confirm the shutdown.
      3. Wait until the computer power supply turns off. Then, switch off the power switch of the printer and monitor.
    11. Maintenance
      1. Maintenance/Service Actions must be performed to ensure that the instrument runs reliably.
      2. Use the Service work area to track and perform service actions. When any service action is due, the Service icon on the navigation bar turns yellow.
      3. Generally, the instrument must be in Standby when you are performing service actions.
      4. To access the Service work area, click the Service button on the navigation bar. The Service work area has the following tabs:
        1. Due:This tab lists service actions that are nearly due or overdue.
        2. All:This tab lists all the service actions.
        3. Begin Of Day:This tab lists service actions that are to be automatically performed at the Begin of Day.
      5. Select the service action to be performed and click Perform. The service action dialog box is displayed.
      6. Follow the instructions listed on the dialog box to perform the service procedure.
      7. Listed below are the Maintenance tasks to be performed daily, weekly, monthly, or on annual basis:
        1. Cobas c501
          1. Daily:
            1. Clean : sample probe & shield pipe
            2. ISE sipper probe
            3. Reagent Probes
            4. Cell Rinse nozzles
            5. Drain port
            6. Perform Daily Pipe
            7. Photometer check reading at 340 nm (Reading < 14000).
          2. Weekly:
            1. Clean Cell Covers
            2. Clean Rinse Stations
            3. Clean IS bath
            4. Perform Weekly Pipe
          3. Monthly maintenance:
            1. Clean incubation water bath
            2. Clean KCI aspiration filter
            3. Clean detergent aspiration filters
            4. Clean circuit board rack filter
            5. Clean power supply filter
            6. Clean radiator filter
            7. Replace reaction cells
          4. Every 2 months:
            1. Replace ISE Measuring Cartridges
          5. Every 6 months:
            1. Replace ISE pinch valve tubing
            2. Replace ISE Sipper tubing
            3. Clean Ultrasonic mixers
            4. Replace ISE Reference Cartridges
            5. Replace Syringe Seals
        2. Cobas e601:
          1. Daily:
          2. Weekly:
            1. Clean PC/CC reservoir fill nozzles.
            2. Clean PC/CC reservoir positions
            3. Clean Pre Clean mixer
            4. Pre Clean separation stations
            5. Clean Incubator
            6. Clean Assay Cup vortex mixer
            7. Clean microbead mixer
            8. Clean Rinse Station
            9. Replace PC/CC reservoirs
            10. Perform Weekly Pipe
          3. Every 2 weeks:
            1. Perform Liquid Flow Path Cleaning
          4. Every 3 months
            1. Replace pinch valve tubing
          5. Every 6 months
            1. Replace Pipettor Seals
    12. Calibration
      1. Calibration materials are made of high quality materials with test system and matrix.
      2. Causes for calibration recommendations:
        1. Automatic calibration at start-up of the analyzer (Start-up calibration).
        2. Automatic calibration at change of reagents:
          1. Calibration at cobas c pack or cobas e pack change
          2. Calibration at lot change
        3. Automatic calibration without reagent change:
          1. Calibration in regular intervals
          2. Calibration in combination with Preventive action
          3. Calibration due to failed QC
        4. Automatic recommendation for recalibration when a calibration failed.
        5. After failed calibration verification.
      3. Calibration procedure:
        1. To request calibration:
          1. Choose Calibration and QC Select in the Work Flow Guide area.
          2. Choose the buttons that are yellow in both the Calibration area and the QC area to select the suggested calibrations and controls for measurement.
          3. Select the list you want to print from the Load List area. Calibration comprises all requested calibrations.
          4. Choose Print to print the selected load list
          5. Load the calibrators and controls on the analyzer as directed by the Calibration Load List.
        2. To load required calibrators:
          1. Using the load list as a guide, prepare all required calibrator and control materials according to the manufacturer's instructions.
          2. Before placing calibrators and controls on the loader, check that no bubbles or foam are visible on the liquid surface.
          3. Load calibrators and controls onto the racks (calibrators onto black racks, controls onto white racks) and place the racks on the loader.
          4. Two or more calibrator racks are to be measured as one group, so do not put other racks between them. When you carry out QC after calibration, QC racks have to follow calibrator racks directly. Do not set other racks with the calibrator and QC racks.
        3. To request calibration for a test manually:
          1. Choose Calibration > Status.
          2. Select the test and reagent (Current or SB) that needs to be calibrated from the list.
          3. Select the appropriate button in the Method area for a Blank, 2 Point, Full, or Span calibration.
          4. Choose Save to save the changes.
          5. Load required calibrators.
        4. To review calibration data:
          1. Choose Calibration > Status.
          2. Select a photometric test from the list on Calibration > Status.
          3. Choose Calibration Result to display the Calibration Result window.
          4. In order to review the results of previous calibrations of the selected test, choose Calibration Trace.
          5. Choose Calibration Result to display the Calibration Result (ISE) window to review ISE calibration.
          6. Select an e 601 test from the list on Calibration > Status, Choose Calibration Result to display the Calibration Result window to review calibration of e601 analyte calibration.
          7. Choose Calibration Trace to open the Calibration Trace window for previous calibration reports.
        5. To measure calibrators:
          1. Choose Start (global button).
          2. Choose Start on the Start Conditions screen. The Start screen closes and the calibration and control run begins.
        6. Validating calibration results.
          1. Failed or newly recommended calibration is indicated on the System Overview screen by the Calibration and QC Select button turning yellow.
          2. Information on the status of the calibrators that are currently processing can be viewed on the Workplace > Calib. Review screen
          3. If a calibration fails or a QC result falls out of the expected range, check for data alarms on the Workplace > Calib. Review screen or on the printouts.
    13. Quality Control:
      1. Regularly perform QC measurements to continually monitor the instrument performance. After measurement of QC samples, data can either be transferred to and processed on a Host or they can be processed on the analyzer.
      2. Working in barcode or non-barcode mode:
        1. On the Utility > System screen under Barcode Setting, you can verify or change whether the system uses barcodes for controls or not.
        2. When working with barcodes, controls are automatically identified.
        3. In case controls are processed in non-barcode mode, it is necessary to assign a rack number and rack position for each control.
        4. The current rack assignment is listed on the QC > Control screen. To provide additional assignments or remove existing assignments choose Rack Assignment. Be sure to use the specified racks and positions. A misplacement would go undetected by the system and yield completely invalid results.
        5. Do not use barcoded controls on a QC rack that is assigned for non-barcoded controls.
      3. Requesting QC measurements:
        1. To perform controls for active reagents:
          1. On QC > Status select the module where the QC is to be performed from the Module box.
          2. If routine QC is to be performed, choose Routine QC to select all tests that are currently loaded on the system and activated for QC. If individual tests are to be selected, go to step 3.
          3. Choose the appropriate test, control and measuring channel, if applicable. Multiple tests, controls and measuring channels can be highlighted.
          4. Choose Select. A blue bar appears in the Selection column. Manual is displayed in the Cause column. The Select button toggles to Deselect.
          5. Choose Save to request the selected controls for measurement.
          6. Print a QC load list and load controls onto the system.
        2. To perform controls for standby reagents:
          1. On QC > Status choose Stand By Bottle QC to display the Stand By Bottle QC window.
          2. Select the appropriate test, control, and measuring channel if applicable. Multiple tests, controls, and measuring channels can be highlighted.
          3. Choose Select. A blue bar appears in the Selection column. Manual is displayed in the Cause column.
          4. The Select button toggles to Deselect
          5. Choose OK to request the selected controls for measurement.
          6. Print a QC Load List and load controls onto the system.
      4. To review quality control results and statistics:
        1. QC > Run Status:
          1. This shows an overview of the last 500 quality control measurements performed on the analyzer or to view the most recent QC measurements for one particular test.
          2. Use the "Test" box to select all tests or one specific test to be displayed on the chart.
          3. The "Run Status" screen consists of a details area in which you will find the control name, the lot number of control and reagent, the test and position and status of reagent, the module, date and time of the run, the limits, results and units.
        2. QC > Individual:
          1. The "Individual" screen lists all non-accumulated QC results.
          2. It also displays details on each tests control, reagent and control lots, the target mean and SD, the number of QC runs carried on the control since last accumulation, the calculated mean and SD, the CV% and most recent control result.
          3. You can also open the "Chart" screen to open individual QC chart window and exclude some qc runs.
              .
            1. To exclude an individual QC point and add a comment:
              1. Select the QC point to be excluded from QC calculations.
              2. Choose Comment on the Individual QC Chart window to display the Comment window.
              3. If necessary type a comment (max. 20 characters) in the Comment box.
              4. Choose Exclude. The window is closed after confirmation.
              5. Choose OK.
      5. To accumulate QC data:
        1. Select the data to be accumulated from the list on the QC > Individual screen.
        2. Choose Accumulate
        3. If Mean-R is set as accumulation method, go to step 4.
        4. In the confirmation window choose Yes to accumulate the selected data.
        5. After accumulation the corresponding data are deleted from the QC > Individual screen and a new standard deviation (SD) and mean value is calculated and displayed on QC > Cumulative.
      6. To assign a control to a specific rack and position:
        1. On QC > Control choose Rack Assignment.
        2. On the Rack Assignment window select the control from the list on the left and an unassigned rack number and position from the list on the right.
        3. Choose Add to register the new assignment.
        4. To remove an existing assignment select the concerned rack number and position from the list on the right and choose Remove.
        5. Choose OK to save the changes and close the window.
      7. To download control data from cobas link:
        1. On the QC > Install screen choose Download.
        2. In the Search Using area select a search option.
        3. Choose Search to start searching for the selected criteria.
        4. Select the check box(es) in the Selection column to download the necessary control information and choose Download.
        5. Confirm the message on the confirmation windows to start the download.
      8. To add controls manually:
        1. Select a blank line from the control list on the QC > Install screen.
        2. Choose Add.
        3. Type all necessary data in the Add Control window
        4. Choose OK. The new control appears on the control list. To run control tests with the newly installed controls, it is necessary to manually enter the target values for each test and then to activate these tests.
        5. Choose Edit to enter target mean, target SD, and confidence limits if necessary Then choose Active Test to activate the tests for the new control.
      9. To edit control values:
        1. On the QC > Install screen choose the name of the control to be edited in the Control list on the left-hand side of the screen.
        2. Choose Edit to display the Edit Control window.
        3. Select the test to be edited from the list.
        4. Type the new target mean and SD (standard deviation) values, cumulative and individual, in the appropriate text boxes.
        5. Choose Calculate to recalculate the confidence limits on the basis of the edited target and SD values and the QC violation rule (set under Utility > Application > Calib.).
        6. Choose Update to update the window.
        7. Repeat steps 3 to 6 for all tests assigned to this control that need to be edited.
        8. Choose OK to save the changes and close the Edit Control window.
      10. To reassign control values for c 501 via download:
        1. Download the control values for controls that need to be reassigned.
      11. To activate control tests:
        1. On the QC > Install screen choose the appropriate control name in the list box on the left side of the screen.
        2. Choose the test to be activated in the list on the right of the screen
        3. Choose Activate Test. The test name highlights in green when the test is activated.
        4. Repeat step 2 and 3 for all tests to be activated for that control.
  12. DOWNTIME PROCEDURES
    1. In case PCS LAB CARE is down, use the downtime form for Biochemistry in reporting results.
    2. In case the test cannot be done, such as in failed QC, unavailability of reagents or consumables, or prolonged instrument downtime, the sample will be sent to PHD-AUH.
